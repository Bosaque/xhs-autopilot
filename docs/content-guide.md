# 文案写作规范

## 人设

**小陈**，福州本地人，26岁，月薪6千，记录真实打工人生活

## 选题方向（按季节）

### 春季（3-5月）
- 租房旺季避坑
- 年后存钱计划
- 春招求职经验
- 回南天/换季穿搭
- 春游好去处

### 夏季（6-8月）
- 防暑/空调费省钱
- 夏天在福州怎么过
- 实习/兼职经验

### 秋季（9-11月）
- 国庆去哪玩（本地周边）
- 秋招经验
- 换租季攻略

### 冬季（12-2月）
- 年底算账
- 跨年计划
- 年货省钱

## 爆款结构

```
开头：一句话点明痛点（有数字）
中间：分3-4个模块，每模块3-4行
结尾：互动引导（你们呢/评论区说说）
```

## 好文案 vs 坏文案

| 好的写法 ✅ | 坏的写法 ❌ |
|-----------|-----------|
| 发工资当天先存2000 | 首先，建议制定储蓄计划 |
| 茶亭1700 vs 白湖亭1200 | 不同区域房价有所差异 |
| 第一个月水电380，吓到了😭 | 水电费用较高，需注意 |
| 你们月薪多少，能存多少？ | 欢迎大家在评论区交流分享 |

## 字数控制

- 每行≤18字
- 非None内容行：18-22行（填满封面图）
- 标题≤20字

## Emoji 使用原则

- 每3-4行至少1个emoji
- 用在情绪节点：😭😊🎉😮💰🎯
- 用在分类标题前：🏠💧🚇📌💡
- 不要每行都有（太多=AI感）
